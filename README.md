### 阅读感受及推荐

​	很不错的书啊，我之前没有接触过操作系统的知识，所以这本书算是我的OS入门书。整个读下来确实是收获颇丰，最起码理解了OS的虚拟化，并发还有持久化这些机制。然后本书的讲解也循序渐进，基本的模式就是：**提出问题 -- 给出方案 -- 新的问题 -- 新的方案**，整体的思路是比较流畅的，也更易于理解和记忆这些机制和实现出现的原因。

​	总之很推荐去读一读
